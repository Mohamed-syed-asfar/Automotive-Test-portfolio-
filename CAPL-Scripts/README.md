# CAPL Scripts

This folder contains CAPL test scripts written for CANoe 
to automate UDS diagnostic testing on automotive ECUs.

## Scripts

| Script | Purpose |
|--------|---------|
| DID_Read_Validation.can | Automated DID read testing using Service 0x22 |
| Session_Control.can | UDS session management testing using Service 0x10 |
| DTC_Read_Clear.can | DTC fault validation using Services 0x19 and 0x14 |
| Security_Access.can | Seed-key security access testing using Service 0x27 |
